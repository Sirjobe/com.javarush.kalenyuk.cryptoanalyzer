Основные задачи:
1. Реализация шифра Цезаря:
1.1. Создание и использование алфавита. +
1.2. Алгоритм сдвига символов в соответствии с заданным ключом. +
2. Обработка файлов:+
2.1. Функциональность для работы с файлами (чтение, запись).+
2.2. Обработка больших текстовых файлов.+
3. Валидация входных данных:+
3.1. Проверка существования файлов. +
3.2. Допустимость ключей. +
4. Режимы работы:
4.1. Шифрование текста:+
Функция шифрования, принимающая файл, ключ и записывающая зашифрованный текст в новый файл.+
4.2. Расшифровка текста:
4.2.1. Функция расшифровки с использованием известного ключа.+
4.2.2. Расшифровка методом brute force (опционально):
Реализация метода brute force для перебора всех ключей до успешного расшифрования.
4.2.3. Расшифровка методом статистического анализа (опционально):
Разработка алгоритма статистического анализа для автоматической расшифровки без ключа,
используя особенности языка.

Дополнительные задачи:
1. Разработка интерфейса пользователя: Текстовое меню или (опционально) графический интерфейс
2. Обработка ошибок и исключений.
3. Оптимизация для производительности.
4. Документация и тестирование.
